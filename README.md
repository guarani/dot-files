### Instalation

Clone the repo:

```
git clone https://github.com/guarani/dotfiles/.git ~/.dotfiles
```

### Vim Configuration

Usage:

```
ln -s ~/.dotfiles/.vimrc ~/.vimrc
```

### Bash Configuration

Usage:

```
ln -s ~/.dotfiles/.bash_profile ~/.bash_profile
source ~/.bash_profile
```

That's it!
